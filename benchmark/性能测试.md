## TCP 协议代理测试

1. 新增一个TCP代理

```text
8005(RemotePort)-->5201(LocalPort)
```

2. 运行目标服务

```shell
iperf3 -s -p 5201
```

3. 正向测试

```shell
iperf3 -c 127.0.0.1 -p 8003   
```

4. 反向测试

```shell
iperf3 -c 127.0.0.1 -p 8003  -R
```
