# 端口池配置
端口池是指预先定义一个可用的端口范围，用于限制系统端口的分配范围，同时用于自动分配端口时的选择范围，避免与系统中其他服务占用的端口冲突。

- 默认范围：[1,65535]
- 限制范围：[1,65535]

```toml
[port_policy]
start = 8100
end = 8109
```

| 参数名 | 类型 | 默认值 | 描述 | 必填 |
| :--- | :--- | :--- | :--- | :--- |
| start | Integer | 1 | 端口范围起始值 | 否 |
| end | Integer | 65535 | 端口范围结束值 | 否 |

:::tip 提示
如果服务端部署在云服务器，还需要在云服务器的**安全组**中开放对应的端口范围。
:::
