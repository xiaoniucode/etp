# 管理面板

管理面板提供可视化界面，用于管理代理配置、客户端和查看监控数据。

```toml
[dashboard]
enabled = true
addr = "localhost"
port = 8020
username = "admin"
password = "123456"
```

## 参数说明

| 参数名 | 类型 | 默认值     | 描述 | 必填     |
| :--- | :--- |:--------| :--- |:-------|
| enabled | Boolean | true    | 是否启用管理面板 | 否      |
| addr | String | 0.0.0.0 | 管理面板绑定地址 | 否      |
| port | Integer | 8020    | 管理面板监听端口 | 否      |
| username | String | -       | 管理员用户名，启用时必填 | 是（启用时） |
| password | String | -       | 管理员密码，启用时必填 | 是（启用时） |
