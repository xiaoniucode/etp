访问令牌用于客户端连接认证，访问令牌包含一个唯一的名称和一个随机生成的字符串。

案例如下：

```toml
[auth]
[[auth.tokens]]
name = "macos"
token = "5E793FAD27AA713D"

[[auth.tokens]]
name = "win"
token = "5E793FAD27AA715D"
```

| 参数名 | 类型 | 默认值 | 描述 | 必填 |
| :--- | :--- | :--- | :--- | :--- |
| name | String | - | 访问令牌名称，唯一标识一个访问令牌 | 是 |
| token | String | - | 访问令牌字符串，随机生成的字符串，长度建议在32位以上 | 是 |


:::info 说明
如果开启了管理面板，也可以直接在管理面板添加访问令牌，此方式为静态配置。
:::
