# 连接重试

当代理客户端与服务端控制连接断开或连接失败时，系统支持自动重试连接。

## 配置示例

```toml
# etpc.toml
[connection]
[connection.retry]
initial_delay = 2
max_delay = 8
max_retries = 5
```

## 参数说明

| 参数名                 | 类型      | 默认值 | 描述        | 必填 |
|:--------------------|:--------|:----|:----------|:---|
| retry.initial_delay | Integer | 1   | 初始重试延迟（秒） | 否  |
| retry.max_delay     | Integer | 20  | 最大延迟时间（秒） | 否  |
| retry.max_retries   | Integer | 5   | 最大重试次数    | 否  |
