如果想要自定义日志记录规则，可以在`etps.toml`或`etpc.toml`中追加下面配置：
```js
[log]
level = "info" #默认为info
path="logs/etps.log" #日志文件路径
pattern = "logs/etps.%d{yyyy-MM-dd}.log" #日志归档格式
```
- level：日志打印级别，默认为INFO，不区分大小写。

:::tip 提示
OFF(关闭) → ERROR(错误) → WARN(警告) → INFO(信息) → DEBUG(调试) → TRACE(追踪) → ALL(全开)
:::
- path：etps主日志保存文件路径，默认为当前路径下`logs/etps.log`

- pattern: 日志归档格式，默认为`logs/etps.%d{yyyy-MM-dd}.log.gz`，每天归档旧日志并压缩保存。
