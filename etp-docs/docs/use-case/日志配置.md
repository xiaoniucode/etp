如果想要自定义日志记录规则，可以在`etps.toml`或`etpc.toml`中添加类似如下配置：

```js
[log]
level = "info"
path = "log"
name = "etps.log"
archivePattern = "etpc.%d{yyyy-MM-dd}.log.gz"
logPattern = "%d{yyyy-MM-dd HH:mm:ss.SSS} [%thread] %-5level %logger{36} - %msg%n"
maxHistory = 30
totalSizeCap = "3GB"
```

- level：日志级别，默认为INFO，不区分大小写。
- path: 日志保存基础路径
- name: 主日志名
- archivePattern: 日志归档格式
- logPattern: 控制台日志输出格式
- maxHistory: 历史日志保存(天)数
- totalSizeCap: 日志总空间大小

:::tip 提示
OFF(关闭) → ERROR(错误) → WARN(警告) → INFO(信息) → DEBUG(调试) → TRACE(追踪) → ALL(全开)
:::
