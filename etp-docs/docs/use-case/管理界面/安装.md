## 快速启动

在`etps.toml`配置文件中添加如下配置，默认面板是关闭状态。

```toml
[dashboard]
enable = true #默认false
addr = "localhost" #默认0.0.0.0
port = 8020 #默认8020
username = "admin"
password = "123456"
```

浏览器访问：http://localhost:8020/login.html

![登录.png](_img/%E7%99%BB%E5%BD%95.png)

## 重置用户名和密钥

在dashboard配置中将`reset`设置为`true`，即可重置用户名和密码，会同步到SQLite数据库，登录以后可以修改。
```toml
[dashboard]
reset=true
```

完整配置如下：
```toml
[dashboard]
enable = true
addr = "0.0.0.0"
port = 8020
username = "新的用户名"
password = "新的密码"
reset = true
```

:::tip 提示
还有一种重置用户名和密码的方式是**删除**当前目录下的`etp.db`数据文件，但是**会删除所有数据**，慎用！
:::
