## 添加映射

先选择一个客户端，属于一个简短的名字标识，协议当前只支持`TCP`，但也支持`HTTP`的代理，公网端口如果不填会自动生成，如果`etps`部署在云服务器上，需要保证分配的端口是开放的。

![添加映射.png](_img/%E6%B7%BB%E5%8A%A0%E6%98%A0%E5%B0%84.png)

## 编辑映射

编辑映射不支持修改映射所在的**客户端**。

![编辑映射.png](_img/%E7%BC%96%E8%BE%91%E6%98%A0%E5%B0%84.png)

## 删除映射

- 会断开与该映射有关的所有公网用户的连接
- 会删除与该端口有关的流量统计记录
- 释放对公网端口的占用，其他映射可以重新使用。

## 状态切换

只有开启状态，公网用户才能与该端口建立连接。如果从开启状态切换为关闭状态，如果之前有公网用户与该端口建立了连接，会断开所有连接，但流量统计记录不会删除，该公网端口依然不能被其他映射使用。
