## 创建客户端

输入一个简短的客户端名字用于标识，名字不能够重复。

![创建客户端.png](_img/%E5%88%9B%E5%BB%BA%E5%AE%A2%E6%88%B7%E7%AB%AF.png)

## 踢掉客户端

- etps与etpc的连接会断开
- 公网用户与该客户端的所有端口建立的所有连接会断开

## 删除客户端

- etps与etpc的连接会断开
- 公网用户与该客户端的所有端口建立的所有连接会断开
- 与该客户端有关的端口映射会被删除
- 流量统计记录会被删除
