多客户端指的是同一个`etps`服务端为多个不同的客户端提供代理服务，对于不同的客户端(client)，端口映射规则是`隔离`的，但`公网端口(remotePort)`必须保证全局唯一，客户端采用`secretKey`唯一标识一个客户端，采用不同密钥可以在同一台设备同时启动多个客户端。

```json
#----------------客户端1----------------#
[[clients]]
name = "windows"
secretKey = "your-secret-key"

[[clients.proxies]]
name = "mysql"
type = "tcp"
localPort = 3306
remotePort = 3307
#status=1

[[clients.proxies]]
name = "web"
type = "tcp"
localPort = 8080
remotePort = 8081

#----------------客户端2----------------#
[[clients]]
name = "macos"
secretKey = "your-secret-key"

[[clients.proxies]]
name = "mysql"
type = "tcp"
localPort = 3306
remotePort = 3308

#----------------客户端...---------------#
```
