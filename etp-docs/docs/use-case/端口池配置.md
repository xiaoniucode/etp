# 端口池配置

从`etp v0.4.0`版本开始，支持在服务端配置文件`etps.toml`中指定公网端口的分配范围(remotePort)，当前版本暂不支持在管理界面中进行动态配置，端口池首次初始化后会被加载到db中，后续以`etp.db`为主，如果需要修改，需要删除db，或者使用SQLite数据库管理工具进行修改（后续完善这个问题）.
- 端口范围默认是：[1024,49151]
- 限制范围：[1,65535]

```toml
[port_range]
start = 8600 #开始端口
end = 8609 #结束端口
```
