# 概述
**etp**(全称：`Easy tunnel proxy`)是一个轻量级的基于**Netty**实现的轻量级内网穿透反向代理应用，即装即用，部署简单，无需安装外部数据库服务，支持**TCP、HTTP**等协议代理，采用**TLS1.3**高性能安全加密协议，保证数据传输的安全性，同时提供图形化管理面板，管理成本低。该应用用于将局域网服务快速暴露为公网服务，供公网用户访问，减少购买昂贵云服务器成本，主要应用场景是：远程桌面、开发与调试、支付接口调试、小程序测试、智能家居与物联网、搭建个人网站或博客等。

### 🌟 功能特性
- 📡 支持**TCP**、**HTTP**等协议
- 🔐 采用**TLS1.3**高性能安全加密协议
- 🍃 支持**Spring Boot&Cloud**生态无缝集成
- 🖥️ 提供易于操作的管理界面
- 🛜 高性能数据传输，延迟低
- 🚀 毫秒级启动
- 🔗 断线重连
- 🔐 身份认证
- 🐒 支持多客户端
- 🧿 自动分配映射端口
- 📄 支持纯Toml配置或管理界面使用
- 📺 跨平台，兼容arm64和amd64架构


### 👥 适合人群

- **开发与运维工程师**：用于日常开发调试、内网穿透、服务暴露、远程联调、SSH等场景。
- **软件测试人员**：快速搭建临时测试环境，实现外部访问内网服务。
- **远程协作与演示**：方便向客户或团队远程演示项目，免去复杂部署，提升效率。
- **物联网/嵌入式开发**者：支持 **ARM** 架构，适用于树莓派等设备远程连接与管理。
- **学生与网络技术爱好者**：适合学习网络编程、代理服务、隧道技术的实践参考工具。
- **初创团队与小微项目**：轻量、易用、低成本，快速实现服务对外开放。
- **云服务使用者**：配合云主机、容器等服务，轻松实现内外网互通与端口映射。
