# 概述
etp（Easy Tunnel Proxy）是一个高性能的内网穿透工具，支持TCP、HTTP协议以及TCP上层协议，支持TLS1.3高效安全加密协议，支持纯Toml静态配置或管理界面动态配置使用，对于Java开发者来说，还可以很方便地和SpringBoot生态集成，无需单独安装客户端即可实现访问公网。

### 功能特性
- 支持TCP、HTTP协议以及TCP上层协议代理
- 主持TLS1.3高效安全加密协议
- 支持Spring Boot&Cloud生态无缝集成
- 内置易于操作的轻量级管理界面
- 高效数据传输，全链路零拷贝设计
- 智能字节数据压缩，大幅度减少带宽
- 毫秒级启动
- 断线重连，保证系统的稳定性
- 隧道级身份认证，避免非法用户连接
- 支持多客户端，同一个服务端管理多个客户端
- 自动分配映射端口，支持端口池
- 支持纯Toml配置或管理界面使用
- 跨平台，兼容arm64和amd64架构


### 适合人群

- **开发与运维工程师**：用于日常开发调试、内网穿透、服务暴露、远程联调、SSH等场景。
- **软件测试人员**：快速搭建临时测试环境，实现外部访问内网服务。
- **远程协作与演示**：方便向客户或团队远程演示项目，免去复杂部署，提升效率。
- **物联网/嵌入式开发**者：支持 ARM 架构，适用于树莓派等设备远程连接与管理。
- **学生与网络技术爱好者**：适合学习网络编程、代理服务、隧道技术的实践参考工具。
- **初创团队与小微项目**：轻量、易用、低成本，快速实现服务对外开放。
- **云服务使用者**：配合云主机、容器等服务，轻松实现内外网互通与端口映射。
