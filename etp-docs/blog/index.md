# 版本历史
详细发布细节请往Github [Release](https://github.com/xiaoniucode/etp/releases) 页面查看

## etp v0.2.0
### ✨ 新功能
- 新增易于操作的图形化管理界面。
- 新增支持日志自定义规则配置。
### ⚙️ 配置变更
- 配置文件中的证书配置属性由 `ssl` 更名为 `tls`。
- 端口映射增加status状态字段，用于控制映射的开启状态。

[查看本次发行完整信息 ➡️](https://github.com/xiaoniucode/etp/releases/tag/v0.2.0)
