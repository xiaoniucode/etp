## 介绍

**etp (Easy Tunnel Proxy)** 是一个高性能的**内网穿透安全网关**。

- 支持 TCP、HTTP协议代理
- 数据压缩
- mTLS 传输加密
- IP CIDR 访问控制（白/黑名单）
- 多用户 Basic Auth 认证
- Token令牌认证，支持多客户端
- 带宽限流与速率限制
- 多种负载均衡策略、故障转移
- 域名、子域名支持
- 内置管理面板
- Spring Boot 集成与 SDK 支持
- 跨平台运行：Windows、Linux、macOS
